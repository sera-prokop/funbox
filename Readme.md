# Инструкция по сборке проекта.

#### 1. Установить «[Yarn](https://yarnpkg.com/en/docs/install)»

#### 2. Установить Gulp CLI
```
$ npm i -g gulp-cli
```

#### 3. Перейти в папку проекта:
```
$ cd путь_к_вашей_папке
```

#### 4. Подтянуть зависимости:
```
$ yarn
```

#### 5. Запустить сборщик:
```
$ gulp
```
